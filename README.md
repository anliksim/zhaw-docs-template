# Introduction

ZHAW SoE Modul [XYZ][1]

## Prerequisites

* node
* npm
* gitbook


## Usage

Clone with submodules

	git clone --recurse-submodules <repository>

Serve book on locahost:4000

	gitbook serve

Build static page

	gitbook build

Create sorted glossary page

	./scripts/build_glossar.sh

Deploy page to gh-pages

	./scripts/deploy.sh
